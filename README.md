### 👗 Stockey - Women's Clothing & Beauty Store

<p align="center">
  <img src="https://github.com/maro-eltma333/maro-eltma333/blob/main/Screenshot%202026-03-03%20222239.png" width="400" height="300" style="margin: 10px; border: 2px solid #FF69B4; border-radius: 10px;">

  <img src="https://github.com/maro-eltma333/maro-eltma333/blob/main/Screenshot%202026-03-03%20222342.png" width="400" height="300" style="margin: 10px; border: 2px solid #FF69B4; border-radius: 10px;">
  <img src="https://github.com/maro-eltma333/maro-eltma333/blob/main/Screenshot%202026-03-03%20222352.png" width="400" height="300" style="margin: 10px; border: 2px solid #FF69B4; border-radius: 10px;">
</p>

💡 **Project Overview**:  
Stockey is a premium web application designed for a high-end women's fashion and cosmetics boutique. It offers a sophisticated design system, real-time administrative tools, and a seamless shopping experience.

🎯 **Key Features**:  
- **Home Page Enhancements**: Special Offer Section and a dedicated Offers Page.
- **Profile Redesign**: Glassmorphism-inspired layout with direct editing and optimized order history.
- **Admin Tools**: Real-time order tracking, inventory management, and site configuration.
- **Store Expansion**: Updated branch network with Arabic addresses and Google Maps links.
- **Security Updates**: Maintenance Mode toggle and enhanced authentication UI.

🔗 **Technology Stack**:  
- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Motion (Framer Motion)
- **State Management**: React Context API
- **Routing**: React Router v7+
