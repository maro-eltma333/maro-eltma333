💡 **Description**:  
Bilingual (EN/AR) sign language learning and translation platform for the Deaf/Hard of Hearing community, featuring 3 custom LSTM models trained with PyTorch for real-time sign recognition (ASL, Arabic SL, ASL Numbers) using optical flow extraction, a 9-table PostgreSQL schema on Supabase with Row Level Security and role-based access, a Text-to-Sign engine with Spotlight Player, Virtual Sign Keyboard, Admin Control Center with analytics dashboards, and an AI backend deployed on Hugging Face Spaces via Docker + FastAPI with the frontend on Vercel.

🔗 **Technology Stack**:  
- **Frontend**: React · TypeScript · Vite · Tailwind CSS  
- **Backend**: FastAPI · PyTorch · OpenCV · Docker  
- **Database**: Supabase (PostgreSQL) · Cloudinary  
- **Deployment**: Vercel · Hugging Face Spaces

💡 **Project Overview**:  
Stockey is a premium web application designed for a high-end women's fashion and cosmetics boutique. It offers a sophisticated design system, real-time administrative tools, and a seamless shopping experience.

🎯 **Key Features**:  
- **Home Page Enhancements**: Special Offer Section and a dedicated Offers Page.
- **Profile Redesign**: Glassmorphism-inspired layout with direct editing and optimized order history.
- **Admin Tools**: Real-time order tracking, inventory management, and site configuration.
- **Store Expansion**: Updated branch network with Arabic addresses and Google Maps links.
- **Security Updates**: Maintenance Mode toggle and enhanced authentication UI.

🔗 **Technology Stack**:  
- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Motion (Framer Motion)
- **State Management**: React Context API
- **Routing**: React Router v7+
